# Github Finder

## Página realizada com React, e utilizando uma api do Github, afim de encontrar perfis e mostrar algumas informações, como número de seguidores, quem a pessoa segue e a foto de perfil

# Funcionalidades

* Busca perfis no Github

# Tecnologias Utilizadas

* ReactJS - Para construção da interface

* JavaScript - Para lógica de programação 

* CSS - Para estilização da interface

* React Hooks - Para gerenciamento de estado

# Conceitos Abordados

* Componentização
* Chamada de API

# Como utilizar

1- Clone o projeto
`git@github.com:helen-andrade/github-finder.git`

2- Instale as dependências
`npm i`

3- Rode o script de desenvolvimento
`npm run dev`
